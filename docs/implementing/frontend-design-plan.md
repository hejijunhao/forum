## Design Philosophy

Forum's frontend embodies three core principles:

1. **Ultra-lightweight**: Minimal chrome, maximum content. No visual clutter.
2. **Calm & readable**: Generous whitespace, muted colors, clear typography.
3. **Agent-ready**: Terminal-first design language that hints at future AI capabilities.

## State Management

### Approach: Server Components + React Query

- **Server Components** for initial data fetching
- **React Query** for client-side caching and mutations
- **URL state** for filters and navigation
- **Local state** for UI interactions (modals, expanded states)
